# react-hook-form and Zod

この記事は **react-hook-form** と **Zod** で作ったフォームのサンプルです。

## 🛠️ 開発環境

- Node.js（18.20.4）
- React（^18）
- Next（14.2.5）
- react-hook-form (^7.52.2)
- Zod（3.23.8）

## 🛹 動作方法

#### 1. リポジトリをクローン

```
git clone https://github.com/twosun-8-git/zod.git
```

#### 2. モジュールをインストール

```
npm i
```

#### 3. 開発環境を起動

```
npm run dev
```
